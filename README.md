# CS50-pset3-Plurality-and-TIdeman

# Plurality Usage
Implement a program that runs a Plurality election.
Run in terminal/CS50 IDE, as below

```
$ ./plurality Alice Bob Charlie
Number of voters: 4
Vote: Alice
Vote: Bob
Vote: Charlie
Vote: Alice
Alice
```
